---

## 오늘의 핵심 목표

오늘의 핵심은 이것이다.

```text
회원가입
-> 로그인
-> JWT 발급
-> 토큰으로 내 정보 조회
-> 토큰으로 게시글 작성
-> 작성자만 수정/삭제
```
