## 오늘 결정할 것

오늘은 배경 지식이 부족해도 결정 가능한 항목부터 닫는다.

- 기술 스택: React, NestJS, FastAPI, PostgreSQL + pgvector
- 핵심 서비스: 운동 기록 인증 게시판
- 핵심 AI 기능: 이전 기록과 현재 기록을 비교해서 다음 운동 목표 추천
- RAG 최소안: 비슷한 과거 운동 기록 검색
- MCP 최소안: 운동명 정규화 또는 외부 운동 정보 조회 도구
- Agent 최소안: 현재 기록 조회, 이전 기록 검색, 분석, 추천 저장까지 이어지는 고정 workflow
- MVP 성공 기준: 로그인 후 운동 기록을 작성하고 AI 분석 결과를 확인할 수 있으면 성공
