# Mobile Workout Session First

The product will prioritize a mobile Workout Session flow over a board-style posting flow. The AI Coach Dashboard's primary job is to move the user into today's workout; the session shows only the exercise name and target set by default, supports quick set checks, lets the user defer an exercise when equipment is occupied, and reveals extra details only on demand. Completed workouts become public workout logs by default, with sharing optional before saving, so the product stays simple during training while still supporting lightweight community reactions afterward.
